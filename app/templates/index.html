Architecture

DevOps & MLOps
CI/CD Pipeline — Jenkins
  • Trigger: GitHub Webhook on push
  • Test: Pytest unit tests auto-run
  • Build: Docker image built & tagged
  • Push: sanjanasshetty/house-price-prediction
  • Deploy: Auto deploy/rollback via docker-compose
AWS Cloud — Terraform
  • Compute: EC2 instances running Docker
  • Storage: S3 buckets for model/data backups
  • Database: RDS PostgreSQL for historical logs
  • Security: IAM Roles & strict Security Groups
Machine Learning
  • Algorithm: Random Forest Regressor (Scikit-Learn)
  • Preprocessing: StandardScaler
  • Serving: Flask REST API /predict
  • Automation: API-triggered retraining script
Observability
  • Prometheus: Scrapes API metrics & latency
  • Grafana: Visualises metrics via provisioning
  • Logging: Real-time feedback via UI terminal
